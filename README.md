## ℹ️ Configuração e Execução
1º Abra o terminal e execute as instruções abaixo:
```Terminal
$ sudo apt-get install cmake libsdl2-dev libsdl2-image-dev libsdl2-mixer-dev libsdl2-ttf-dev
```
2º Em ```game-template```, digite: 
```Terminal
$ cmake .
```
3º Digite:
```Terminal
$ make
```
4º Digite:
```Terminal 
./myawesomegame
```
